Test 2

El fragmento de código de nuestro fichero `test.js` nos devuelve un output no deseado. Queremos imprimir un valor incremental a cada segundo pero lo que nos devuelve el código es el mismo valor en cada iteración.

1. Sin necesidad de ejecutar el código, ¿sabrías decirnos qué valor imprimiría por consola el script?

> 5
> 5
> 5
> 5
> 5

¿Cuál es el motivo?

La variable "i" es global a la ejecución. El for se ejecuta y define 5 timeouts que imprimen "i", pero cuando se ejecuta la función del timeout, ya el valor de "i" es igual a 5.

2. Sabiendo que el output que buscamos es el que encuentras bajo estas líneas… ¿Cómo solucionarías el fragmento de código para que el output sea el deseado?

Declaramos una variable local al for, le asignamos el valor de i, y esa variable es la que le pasamos a la función del timeout.

            
for (var i = 0; i < 5; i++) {
    let localIndex = i;
    setTimeout(function () {
        console.log(localIndex);
    }, 1000);
}
